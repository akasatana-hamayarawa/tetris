# テトリス完全版

ブラウザだけで遊べる、多機能な対戦テトリスです。

## 特徴

- **対人戦(オンライン対戦)** — ルームコードを作成/入力するだけで、ブラウザ同士がP2Pで直接接続して対戦できます
- **CPU対戦** — 一人でも対戦の感覚を練習できます
- **一人でプレイ(ソロ)** — レベルが上がっていく耐久モード
- **テンプレートモード** — DT砲・パフェ・REN系・エレベーター系など、定番の積みパターンを練習できます
- **リプレイ機能** — 対戦を振り返って見返せます
- **チュートリアル** — 初めての方向けの操作ガイド
- **セーブ機能** — 複数スロットにゲームの状態を保存・再開できます
- **T-Spin / Back-to-Back / Combo / Perfect Clear** 判定に対応
- **8種類のテーマ** — Default / Retro / Light / Dark / Cyberpunk / Soft / あんぽんたん / Minimal
  設定画面からいつでも切り替え可能
- **日本語 / English** 表示切り替え対応
- キーボード・ゲームパッド・タッチ操作に対応

## 遊び方

### デフォルト操作

| 操作 | キー |
|---|---|
| 左移動 | ← |
| 右移動 | → |
| ソフトドロップ | ↓ |
| ハードドロップ | Space |
| 右回転 | ↑ |
| 左回転 | Z |
| ホールド(メイン) | C |
| ホールド(サブ) | Shift |
| ポーズ | Escape |
| リトライ | R |

キー配置・DAS/ARR/SDFなどの操作感覚は、設定画面からすべて変更できます。

## プロジェクト構成

```
テトリス完全版/
├── index.html          # エントリーポイント
├── script.js            # ゲーム本体ロジック(移行中。詳細は下記「開発状況」参照)
├── style.css             # 基本スタイル
├── theme/                # テーマ(フォルダを追加するだけで新テーマを認識する設計へ移行中)
│   └── <テーマ名>/
│       ├── theme.json     # テーマのメタ情報・配色
│       ├── theme.css       # テーマ専用の装飾CSS
│       └── assets/          # テーマ専用の画像等
├── assets/                # 画像・音声・フォントなどの共通アセット
│   └── audio/               # BGM・効果音置き場(README.md に命名規則あり。現在は未収録)
├── src/                   # 新アーキテクチャへの移行先(進行中)
│   ├── core/                # EventBus, Logger, Constants
│   └── managers/            # SaveManager, SettingsManager, AssetManager, AudioManager, ThemeManager など
├── template/              # HTMLテンプレート
└── tools/                 # 開発用チェックスクリプト
```
